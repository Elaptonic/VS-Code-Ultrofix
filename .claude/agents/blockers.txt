## Frontend Task Dependencies
- Any screen/hook consuming a new or changed API endpoint depends on:
  1. Schema change in lib/db/src/schema/ (Backend Agent)
  2. lib/api-spec/openapi.yaml updated + `pnpm --filter @workspace/api-spec run codegen` run
  3. lib/api-zod and lib/api-client-react regenerated and typechecked

## Backend Task Dependencies
- Any route change that affects the API contract depends on:
  1. lib/api-spec/openapi.yaml updated to match the new request/response shape
  2. Codegen run so lib/api-zod stays in sync (Frontend Agent consumes this)
